# Data Quality Assessment & Preprocessing

## Project Overview
This repository demonstrates foundational data cleaning and exploratory data analysis (EDA) techniques essential for any Machine Learning pipeline. Using a raw sales dataset, it covers identifying anomalies, handling missing values, standardizing data, and applying dimensionality reduction.

## Key Techniques Applied
* **Data Quality Assessment:** Identifying incorrect data types and structural issues.
* **Missing Value Imputation:** Applying median imputation to preserve data integrity against extreme outliers.
* **Outlier Detection:** Utilizing the Interquartile Range (IQR) method.
* **Feature Scaling:** Implementing Min-Max and Z-score normalization for distance-based ML models.
* **Dimensionality Reduction:** Applying Principal Component Analysis (PCA) to evaluate feature variance.

## Technologies Used
* Python
* Pandas, NumPy
* Scikit-Learn
* Matplotlib, Seaborn
